### Draft an Executive Update

```
Draft an exec update on [project] for this week
```

- Progress update
- Key metrics
- Blockers and risks

### Write a Team Update

```
Write an update for the team on [project/sprint]
```

**What it does**:
- Celebrates wins
- Shares progress
- Calls out upcoming work
- More casual tone than exec updates

**When to use**: Sprint reviews, weekly team updates

---

### Create Stakeholder Comms

```
Draft an announcement about [change/launch] for [audience]
```

**What it does**:
- Clear, concise messaging
- Addresses concerns proactively
- Includes FAQs
- Tailored to audience

**When to use**: Feature launches, process changes

---

### Prepare for Stakeholder Meeting

```
Help me prepare for a meeting with [stakeholder] about [topic]
```

**What it does**:
- Creates agenda
- Prepares talking points
- Anticipates questions
- Defines success criteria for meeting

**When to use**: Before important meetings

---

### Generate FAQ

```
Create an FAQ for [feature/change]
```

**What it does**:
- Lists likely questions
- Provides clear answers
- Organized by category
- Includes examples

**When to use**: Launches, rollouts, changes
