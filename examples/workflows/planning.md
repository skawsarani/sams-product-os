### What Should I Work On?

```
What should I work on today based on our priorities?
```

**What it does**:
- Reviews P0/P1 items
- Checks deadlines and dependencies
- Suggests focus areas
- Considers team capacity

**When to use**: Daily planning

---

### Prioritize These Items

```
Help me prioritize these items: [paste list or reference file]
```

**What it does**:
- Applies prioritization framework (RICE, Impact/Effort, etc.)
- References product strategy
- Provides rationale for each priority

**When to use**: Planning sessions, roadmap reviews

---

### Create a Roadmap

```
Create a quarterly roadmap based on @knowledge/product-strategy/
```

**What it does**:
- Organizes initiatives by quarter
- Aligns with strategic goals
- Shows dependencies and milestones

**When to use**: Quarterly planning

---

### Weekly Planning

```
Help me plan this week's work
```

**What it does**:
1. Reviews last week's progress
2. Checks roadmap priorities
3. Considers upcoming deadlines
4. Suggests weekly focus areas
5. Creates weekly plan document

**When to use**: Monday mornings

---

### Sprint Planning

```
Help me plan the next sprint based on our backlog
```

**What it does**:
- Pulls from prioritized backlog
- Estimates effort
- Balances new features, bugs, tech debt
- Creates sprint goals

**When to use**: Sprint planning meetings
