### Apply a Framework

```
Apply [framework name: RICE, Jobs-to-be-Done, etc.] to [item/feature]
```

**What it does**:
- Uses PM framework for analysis
- Shows scoring or breakdown
- Provides insights and recommendations

**When to use**: Prioritization, feature design, strategy

---

### Generate a Template Library

```
Review my recent docs and suggest templates we should create
```

**What it does**:
- Analyzes document patterns
- Identifies repeatable formats
- Suggests template creation
- Prioritizes by usage frequency

**When to use**: Quarterly template review
