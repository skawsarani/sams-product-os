## Triage Bugs

### Prioritize Bugs

```
Help me prioritize these bugs: @backlog/bugs.md
```

**What it does**:
- Categorizes by severity (Critical/High/Medium/Low)
- Assesses user impact
- Considers workarounds
- Recommends fix timeline

**When to use**: Bug triage

---

### Create a Bug Report

```
Create a detailed bug report for [issue description]
```

**What it does**:
- Structured report (steps to reproduce, expected vs. actual, impact)
- Assigns severity
- Suggests root cause areas
- Includes acceptance criteria for fix

**When to use**: Escalating bugs to engineering

---

### Incident Post-Mortem

```
Help me write a post-mortem for [incident]
```

**What it does**:
- Timeline of events
- Root cause analysis
- Impact assessment
- Action items to prevent recurrence
- Blameless format

**When to use**: After incidents or outages
