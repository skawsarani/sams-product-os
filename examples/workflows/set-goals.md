## Set Goals

### Set OKRs

```
Help me create OKRs for [team/product] for [timeframe]
```

**What it does**:
- Defines Objectives (qualitative goals)
- Creates Key Results (measurable outcomes)
- Aligns with company strategy
- Suggests initiatives to achieve them

**When to use**: Quarterly/annual planning

---

### Align to Strategy

```
Check if [feature/initiative] aligns with our product strategy
```

**What it does**:
- References `knowledge/product-strategy/`
- Assesses strategic fit
- Identifies conflicts or gaps
- Provides recommendation

**When to use**: Before committing resources

---

### Create a Vision Doc

```
Help me create a product vision document for [product/area]
```

**What it does**:
- Defines long-term vision (2-3 years)
- Explains the "why"
- Describes target customers and problems
- Outlines success criteria

**When to use**: New product, strategic reset
