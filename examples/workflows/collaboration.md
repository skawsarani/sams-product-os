### Prepare Handoff

```
Create a handoff document for [project/area] to [person/team]
```

**What it does**:
- Current state summary
- Key decisions and context
- Open questions
- Next steps and recommendations

**When to use**: Transitions, vacations

### Cross-Functional Alignment

```
Create an alignment doc for [feature] across [teams: eng, design, marketing]
```

**What it does**:
- Shared understanding of goals
- Each team's responsibilities
- Dependencies and timeline
- Success criteria

**When to use**: Cross-functional projects
