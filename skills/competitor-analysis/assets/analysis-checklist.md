# Competitive Research Analysis Checklist

Use this checklist before finalizing a competitor analysis report to ensure comprehensive, high-quality research.

---

## Completeness Check

### Required Information Gathered

- [ ] **Company overview** documented (founding, size, mission, recent developments)
- [ ] **Product features** listed and analyzed (minimum 5 key features)
- [ ] **Pricing model** documented (at least 2 tiers with details)
- [ ] **Target customers** identified (ICP, named customers, or use cases)
- [ ] **Strengths** documented (minimum 3 with evidence)
- [ ] **Weaknesses** documented (minimum 3 with evidence)
- [ ] **Customer testimonials** included (minimum 3 quotes - mix of positive and critical)
- [ ] **Market positioning** analyzed (value prop, messaging themes)
- [ ] **Strategic implications** identified (threats + opportunities)
- [ ] **Data sources** cited for all claims

---

## Quality Check

### Evidence & Citations

- [ ] All major claims trace to sources (URLs, file paths, or "User-provided")
- [ ] All testimonials are verbatim quotes (not paraphrased)
- [ ] All testimonials have attribution (name/role or "Anonymous", source, date)
- [ ] All web sources include access date
- [ ] Pricing information is current (<6 months old) or flagged as potentially outdated
- [ ] Important claims cross-referenced across 2+ sources

### Analysis Depth

- [ ] Strengths include "Why it matters" and "Threat to us" sections
- [ ] Weaknesses include "Opportunity for us" section
- [ ] Feature comparison includes "Comparison to Us" column (if product-info.md exists)
- [ ] Strategic implications section is actionable (not just descriptive)
- [ ] Executive summary captures 2-3 key strategic insights (not just facts)

### Objectivity & Balance

- [ ] Both strengths AND weaknesses documented (not just one)
- [ ] Positive AND critical testimonials included
- [ ] Avoided confirmation bias (didn't only look for weaknesses)
- [ ] Acknowledged competitor strengths honestly
- [ ] Distinguished facts from assumptions
- [ ] Noted conflicting information when found

---

## Source Quality Check

### Multi-Source Validation

- [ ] Pricing validated from official website (primary source)
- [ ] Key features validated on product page + customer reviews
- [ ] Company size/funding validated from 2+ sources (LinkedIn, Crunchbase, etc.)
- [ ] Strengths/weaknesses confirmed across multiple reviews (not single-source)

### Source Credibility

- [ ] Prioritized primary sources (official website, verified reviews)
- [ ] Flagged low-credibility sources (anonymous posts, outdated articles)
- [ ] Cross-referenced marketing claims with customer feedback
- [ ] Avoided relying solely on competitor's marketing materials

---

## Data Freshness Check

- [ ] Data collection date noted in "Last Updated" field
- [ ] Pricing information is <6 months old OR flagged as potentially outdated
- [ ] Review quotes include dates (to assess recency)
- [ ] Company information (size, funding) is <12 months old
- [ ] Noted when information couldn't be verified as current
- [ ] Recommended re-research timeline included (typically 3-6 months)

---

## Formatting Check

- [ ] Tables used for features comparison (easier to scan)
- [ ] Tables used for pricing tiers (easier to scan)
- [ ] Blockquotes used for testimonials (proper formatting)
- [ ] ✅/⚠️/❌ symbols used in feature comparison (if comparing to our product)
- [ ] Section headers follow template structure
- [ ] Markdown links work (no broken URLs)
- [ ] File saved with correct naming: `competitor-[name]-comparison.md` (lowercase-with-hyphens)

---

## Gap Identification Check

- [ ] Missing information explicitly flagged (e.g., "Pricing not publicly available")
- [ ] Data gaps noted in "Notes & Caveats" section
- [ ] Outdated information flagged with dates
- [ ] Single-source claims noted when important
- [ ] Conflicting information documented
- [ ] Suggested next steps for filling gaps (user interviews, sales calls, etc.)

---

## Strategic Value Check

- [ ] Executive summary provides actionable insights (not just facts)
- [ ] Strategic implications section answers "So what?" question
- [ ] Threats section identifies real risks to our business
- [ ] Opportunities section identifies exploitable gaps
- [ ] Recommended response section provides concrete next steps
- [ ] Report helps PM make strategic decisions (not just informational)

---

## Final Review

Before marking research as complete:

- [ ] Read executive summary - does it capture the most important findings?
- [ ] Scan strengths - would competitor agree these are their advantages?
- [ ] Scan weaknesses - are these validated by customer feedback?
- [ ] Check testimonials - are they verbatim with proper attribution?
- [ ] Review sources - can someone verify all claims?
- [ ] Read strategic implications - are recommendations actionable?

---

## Common Mistakes to Avoid

### ❌ Incomplete Analysis
- Only listing features without analysis
- Missing pricing or testimonials
- No strategic implications
- Vague or generic conclusions

### ❌ Poor Evidence
- Paraphrasing testimonials instead of quoting
- Missing sources for claims
- Single-source for important information
- No dates on sources

### ❌ Lack of Objectivity
- Only focusing on weaknesses (ignoring strengths)
- Underestimating competitor capabilities
- Confirmation bias (only finding what you expected)
- Dismissing competitor strengths as "marketing spin"

### ❌ Outdated Data
- Using pricing from 2+ years ago
- Relying on old reviews
- Not noting when data was collected
- Assuming information is current without verification

### ❌ Missing Strategic Value
- Just listing facts without analysis
- No "So what?" insights
- Missing threats and opportunities
- No actionable recommendations

---

## Checklist Summary

**Minimum Requirements to Consider Research Complete:**

✅ **Coverage**: All 8 sections documented (company, features, pricing, customers, strengths, weaknesses, testimonials, positioning)

✅ **Evidence**: Every section has citations; testimonials are verbatim quotes

✅ **Quality**: Cross-referenced important claims; balanced strengths and weaknesses

✅ **Freshness**: Noted data currency; flagged outdated information

✅ **Strategic**: Executive summary + strategic implications provide actionable insights

✅ **Format**: Tables for features/pricing; proper markdown formatting

If all checked ✅, the research report is ready to save and deliver to user.
